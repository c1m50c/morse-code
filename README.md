# **morse-code**
Morse Code Encoder & Decoder written in Python.


## **Installing Requirements**
```bash
$ cd morse-code
$ pip3 install -r ./requirements.txt
```


## **Running**
```bash
$ cd morse-code
$ python3 src/main.py
...
```
